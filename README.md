# 📊 AI Data Analyst Agent

An AI-powered data analysis web application built using **Flask + Python** that allows users to upload CSV files, generate insights, visualize data through charts, and perform intelligent analysis.

---

## 🌐 Live Demo
👉 https://ai-data-analyst-agent-1-fhs2.onrender.com/

---

## 🧠 Project Features

- 📁 Upload CSV datasets easily  
- 📊 Automatic data analysis  
- 📈 Interactive charts & visualizations (Matplotlib)  
- 🤖 AI-based insights generation  
- ⚡ Fast and simple Flask web interface  
- 📉 Data cleaning and preprocessing support  
- 🧾 Structured reporting of results  

---

## 🛠️ Tech Stack

- Python 🐍  
- Flask 🌐  
- Pandas 📊  
- Matplotlib 📈  
- Gunicorn 🚀
